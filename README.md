# Basic project

## Blueprint project with Flash backend and MithrilJS frontend. Button generating random number from 1 to 10.

### Structure:

```
random-number-app/
├── README.md
├── Pipfile
├── app.py
├── static/
│   ├── css/
│   │   └── style.css
│   └── js/
│       ├── mithril.js
│       └── app.js
└── templates/
    └── index.html
```

### Install:

```
pipenv install
```

### Run:

```
pipenv run flask run
```
